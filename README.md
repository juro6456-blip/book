# Living Book

Красивый сайт-книга с перелистыванием и настоящей серверной админкой.

## Запуск

```powershell
npm start
```

После запуска:

- книга: http://localhost:3000
- админка: http://localhost:3000/admin

Изменения из админки сохраняются в `data/book.json` и видны всем, кто открывает этот же сервер. Открытые страницы книги сами подтягивают обновления примерно каждые 3.5 секунды.

## Пароль для админки

Локально админка открыта без пароля. Для настоящего сайта запустите сервер с паролем:

```powershell
$env:ADMIN_PASSWORD="your-strong-password"
npm start
```

Логин в окне браузера может быть любым, пароль должен совпадать с `ADMIN_PASSWORD`.

## Что умеет админка

- менять название, автора, подзаголовок и текст на обложке;
- добавлять, удалять и редактировать главы;
- писать эпиграфы, основной текст и заметки автора;
- включать большую заглавную букву;
- менять тему, бумагу, размер шрифта и расстояние между строками;
- автоматически сохранять изменения на сервер.
